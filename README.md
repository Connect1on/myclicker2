# myclicker2

Игра на тему кликер
Содержит саму игру а так же папку где можно редактировать и добавлять новый контент, с учётом что у вас есть gamemaker
Имейте ввиду что это лишь папка разработки, внутри программы нужно сделать сборку игры и уже добавлять дату и екзешник игры в корневую папку, заменяя старое
Игра запускается через екзешник и не требует сборки докера
